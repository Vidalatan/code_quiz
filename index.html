Code Quiz

Rules

  • Questions will be multiple choice, or true/false
  • Each question will have it's own timer, the length of which depends on the amount of available answer choices. If that timer reaches zero (0) the quiz will end and your score will be entered as is
  • Score is calculated by adding 20 points for each question, plus however many seconds remain on the timer for each successfully answered question.
  • You will be prompted for a username when you begin, if you don't select a username then should you choose to save your score it will overwrite the previous anonymous user the next time you revisit the page.
